# Binary-Search-Tree

This is my own implementation of a binary search tree in C. The methods I included were:

create() - Creates a new tree with the root value
add() - adds value to tree
delete() - deletes value from tree
search() - searches for value in tree and returns the node
delete_tree() - deletes the whole tree and all of its children
inorder_traversal() - prints the tree using In-order traversal (left -> node -> right)
preorder_traversal() - prints the tree using Pre-order traversal (node -> left -> right)
postorder_traversal() - prints the tree using Post-order traversal (left -> right -> node)
